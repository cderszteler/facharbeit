Hauptteil:
  1. Benötigtes Vorwissen
    1.1 Komplexe Zahlen ("Benötigtes (Vor-)Wissen", "voraussetzungen")
      =>
        - Addition + Quadrieren
        - Complex (plane)
          - Unterschied zwischen x/y Graph
            => x/y sind nur reelle Zahlen, zwei unterschiedliche Zahlen bzw. Variablen also. Z.B. x (1. Variable) die in f(x) eingesetzt wird (und somit y als 2. Variable)
            => c hingegen ist komplex, es handelt sich also um eine Zahl, die jedoch in zwei Teile aufgebrochen werden kann
              -> a + bi (1. Teil: a, 2. Teil: b)
    1.2 *Iteration*, was bedeutet das? | Mandelbrot-Menge interessiert daran, wie diese Iterationen verlaufen (explodieren/"verlaufen gegen Unendlich", bleiben beschränkt/"(verlaufen gegen Null", etc.)
      => Tabelle von:
        - fc(z) = z^2 + c, bei c = 1 <== Explodiert
        - fc(z) = z^2 + c, c = -1 <== Explodiert
      -> Wurde herausgefunden, dass jede komplexe Nummer außerhalb von Radius 2 immer explodiert
  2. Mandelbrot-Menge
    2.1 Vorzeigen der Definition der Mandelbrot-Menge "M = ..."
      - Konkret für Mandelbrot |s.o.|: (1) Explodiert (gegen Unendlich) (2) Bleibt im Bereich <= 2
        -> Lediglich 2. Fall interessant, genauer: Wie viele Iterationen notwendig
    2.2 Visuelle/Grafische Darstellung einer Mandelbrot-Menge analysieren | Überleitung durch: "Wie sieht sowas aus, etc."
      => "Gefärbter"/"Sichtbarer" Teil der Mandelbrot-Menge beschäftigt sich mit (2) Fall
      => Farbe gibt an, wie viele Iterationen notwendig sind, um herauszufinden, dass die Zahl explodiert
            -> schwarz bedeutet: keine explosion (z.b. nicht nach 200 iterationen)
